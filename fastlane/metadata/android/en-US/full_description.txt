Remap the Nothing Phone Essential Key to your own actions — no root required.

Single, double, triple, and long press each get their own action: launch an app,
toggle the flashlight, take a screenshot, lock the screen, play/pause media, or cycle
the ringer mode.

Features:

* Four gestures, four independent actions.
* "Learn your key" setup captures the key's scan code at runtime, so it adapts to any
  model or firmware.
* Guided accessibility-service enablement through Android Settings.
* Single-press unlock wizard: freeing the single press requires Nothing's Essential
  Space packages to be disabled. The wizard opens each package's App Info page for
  manual disable or restore and shows live per-package status. Double, triple, and
  long press work without this step.
* Nothing OS design language: pure-black canvas, hairline cards, monospaced labels.

Requires a Nothing phone with the Essential Key, running Android 15 or newer.

Essential Key Tools needs the Android AccessibilityService API for its core key-remapping feature.
The service receives Essential Key hardware key events when the app is not open, recognizes single,
double, triple, and long presses, and runs only the fixed shortcut you assigned.

While the service is enabled, Essential Key Tools accesses key codes, scan codes, press or release
state, and event time. It processes this data on your device in real time to recognize those
gestures. Screenshot and Lock screen mappings use the corresponding Android accessibility system
actions. The service follows only the fixed gesture-to-action mapping you configure; it does not
choose or plan actions.

Hardware key-event details are not stored, transmitted, or shared with the developer or third
parties. Only the scan code you confirm in Key Setup is stored on your device. The app does not read
screen content, text fields, or typed text.
